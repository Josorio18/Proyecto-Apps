# UrbanSteps - Aplicación de Calzado (Android)

UrbanSteps es una aplicación móvil robusta desarrollada para el ecosistema moderno de Android, especializada en la venta de calzado deportivo y casual (estilo Nike).

## Características Técnicas

*   **IDE**: Android Studio (Ladybug).
*   **Framework de UI**: Jetpack Compose (100% declarativo).
*   **Lenguaje**: Kotlin.
*   **Arquitectura**: MVVM (Model-View-ViewModel).
*   **Gestión de Datos**: Room Database para persistencia local.
*   **Navegación**: Jetpack Navigation Component.
*   **Diseño**: Material Design 3 con tema personalizado UrbanSteps.

## Requisitos Cumplidos

1.  **Interfaz y Layouts (>80%)**: Implementación fiel de los prototipos en Figma, incluyendo Splash Screen, Catálogo, Detalle de Producto y Carrito.
2.  **Navegación (>70%)**: Flujo funcional entre todas las pantallas principales utilizando un NavHost centralizado.
3.  **Almacenamiento Local (>30%)**: Persistencia de productos favoritos y gestión completa del carrito de compras mediante Room.

## Instalación y Ejecución

1.  Clonar el repositorio:
    ```bash
    git clone [URL_DEL_REPOSITORIO]
    ```
2.  Abrir el proyecto en **Android Studio**.
3.  Sincronizar el proyecto con los archivos de Gradle.
4.  Ejecutar en un Emulador o dispositivo físico (API 24+).
